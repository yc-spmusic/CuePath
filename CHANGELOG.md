# 更新日誌 (CHANGELOG)

所有對 CuePath 的重要更新都會記錄在此檔案中。

## [0.1.1-BETA] - 2026-04-29

### ✨ 新增功能
- **A/B Loop 拖曳**：現在可以直接在時間軸上拖曳 A 與 B 標記點來調整循環區間。

### 🎨 優化與調整
- **字體更換**：切換為系統內建的可商用黑體組合（system-ui），確保法律安全與各平台美觀。
- **版本標示**：在頂部導覽列加入版本號顯示。

---

## [0.1.0-BETA] - 2026-04-29

### 🚀 初始版本發布
- 支援基礎音檔播放與波形顯示。
- 支援 Marker 與 Region 標記功能。
- 支援專案匯入/匯出 (.cuepath)。
- 支援 CSV 與 Markdown 匯出。
- 支援 Windows YouTube 輔助下載腳本。
- 支援 localStorage 自動儲存。
